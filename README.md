# 📊 Dashboard Web

Dashboard administrativo desenvolvido com HTML, CSS e JavaScript.

## ✨ Funcionalidades

* Menu lateral de navegação
* Cards de estatísticas
* Gráficos interativos com Chart.js
* Interface moderna em tema escuro
* Layout responsivo

## 🚀 Tecnologias

* HTML5
* CSS3
* JavaScript
* Chart.js
* Font Awesome

## 📷 Demonstração

Painel administrativo com indicadores de desempenho, estatísticas e visualização de dados em gráficos.

## 📁 Estrutura

```bash
Dashboard/
├── index.html
├── Style/
│   └── index.css
├── Script/
│   └── script.js
└── README.md
```
